# 理想光学系统物象关系图解演示程序

直接用浏览器打开 `index.html` 即可运行，无需安装或联网。

功能包括：会聚/发散系统切换、参数设置、拖动物体和焦点、分步特殊光线作图、自动演示、公式计算、实虚/正倒/大小判断，以及像在无穷远时的专门提示。

本次增加了三个教学辅助功能：

- “虚像：H—H′内”预设，展示虚像在 H 的右侧、H′的左侧的情况；
- “突出显示经过 F 的辅助线”，以高对比虚线和焦点光环强调第二条特殊光线的构造；
- 像区放大滑块，放大时自动以像与光轴之间的区域为中心；可一键回到完整视图。

也支持虚物（`l > 0`）：可用“物的类型”切换，或直接选择“虚物：H—H′内”“虚物：H′右侧”预设。虚物用橙色虚线箭头表示；橙色虚线光路表示实际入射光束延长后将会聚到虚物的位置。H 与 H′之间的等高传递路径已保留显示，不再省略。

“当前模式”可切换为“由物求像”或“由像求物”。在“由像求物”模式中，输入 `l′`、`y′` 及其正负属性，程序依据 `1/l = 1/l′ − 1/f′` 和 `y = y′/β` 反求物距、物高；给定像会以橙色突出显示，且可在图中左右拖动。

符号约定与设计文稿一致：光线从左向右传播为正，物距为 `l`，像距为 `l′`，横向放大率为 `β`，孔径角保留为 `u`（本基础版不参与计算）。
