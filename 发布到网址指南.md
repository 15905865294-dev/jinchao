# 发布成可分享的网址

这个程序是纯静态网页，不需要服务器程序。发布后，别人只要点击网址就能使用，不需要收到文件。

## 推荐：GitHub Pages

1. 注册并登录 GitHub；
2. 新建一个公开仓库，例如 `ideal-optics-visualizer`；
3. 上传本文件夹中的 `index.html`（以及 `README.md`）；
4. 在仓库的 **Settings → Pages** 中，将发布来源设为 `main` 分支、根目录；
5. 保存并等待部署完成。GitHub 会显示一个形如 `https://你的用户名.github.io/ideal-optics-visualizer/` 的网址；
6. 把该网址发给同学或老师即可。

## 需要我协助时

请先在浏览器登录你的 GitHub 账号，并告诉我你准备使用 GitHub Pages。我可以继续协助你建立仓库、上传页面并确认最终网址；在公开发布前会向你确认仓库名称与发布范围。
